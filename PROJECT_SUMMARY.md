## ğŸ‡ºğŸ‡¸ English Summary

**AI EC SEO Booster** is an AI-powered e-commerce market analysis and SEO content strategy generation tool driven by Google Gemini AI. Designed specifically for e-commerce professionals, it automates the entire workflow from product information input to professional landing page prompt generation through a three-stage progressive process: market analysis, competitor research, buyer persona creation, content strategy planning, and SEO optimization recommendations.

### Core Value

- **Time Savings**: Reduces hours of market analysis and strategy planning work to just minutes
- **Professional Analysis**: AI automatically generates comprehensive reports including product core value, market positioning, competitor analysis, and buyer personas
- **SEO Optimization**: Provides professional SEO guidance for each content topic, including keyword density, semantic keywords, and linking strategies
- **One-Click Prompt Generation**: Generates detailed landing page code prompts for Google AI Studio, quickly producing complete React + Tailwind CSS code

### Technical Features

- Built with React 19 + TypeScript for type safety and code quality
- API Key management via Context API, stored only in browser localStorage for security
- Supports product image uploads with Gemini Vision API for visual analysis
- Responsive design, perfectly supporting desktop, tablet, and mobile devices
- One-click download of analysis reports and content strategies in Markdown format
